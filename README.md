Simple vampire survivors clone game with multiplayer made with photon fusion plugin.
This project was created as part of my learning path and first acquaintance with Photon Fusion 2. Feel free to use it as learning example, or maybe for something else. 

Tech stack: Unity, Photon Fusion 2 client/host mode, VContainer, Firebase analytics, UniTask

Art assets used in this project: https://sethbb.itch.io/32rogues
